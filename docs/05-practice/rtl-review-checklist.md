# RTL Review Checklist（草案）

与 [01-rtl/08-coding-guidelines.md](../01-rtl/08-coding-guidelines.md) 配合使用。

## 结构与时序

- [ ] 模块端口位宽与顶层/约束一致
- [ ] 每个时钟域有明确复位策略
- [ ] 异步复位是否同步释放（若项目要求）
- [ ] 无门控时钟伪写法；使能寄存器合法
- [ ] 无组合环路

## 组合逻辑

- [ ] 所有 `always_comb` / `@(*)` 对输出完整赋值
- [ ] 综合无意外 latch 警告
- [ ] `case` 有 `default` 或 `unique` 覆盖

## CDC

- [ ] 跨时钟域路径已标注
- [ ] 单 bit / 多 bit / 数据总线使用合适同步结构
- [ ] 无跨域直接组合逻辑

## 资源

- [ ] 乘法/除法/ROM 规模在预算内
- [ ] RAM 读写时序符合目标器件推断模板

## 可维护性

- [ ] 命名、文件组织符合项目规范
- [ ] 魔数已参数化
- [ ] 仿真专用代码已 `ifdef` 隔离
